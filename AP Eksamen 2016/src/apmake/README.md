# APMake Skeleton Overview

* `Ast.hs` - The abstract syntax tree

## Parser Skeleton

* `Parser.hs` - Wrapper, exporting just the functions we asked for.
* `Parser/Impl.hs` - For your actual implementation.
* `Parser/WhiteBoxTests.hs` - For your internal tests.
* `Parser/BlackBoxTests.hs` - For your exported API tests.
* `SimpleParse.hs` - `SimpleParse` from the course, with a couple minor additions.

## Interpreter Skeleton

* `Interp.hs` - Wrapper, exporting just the functions we asked for.
* `Interp/Impl.hs` - For your actual implementation.
* `Interp/WhiteBoxTests.hs` - For your internal tests.
* `Interp/BlackBoxTests.hs` - For your exported API tests.
